# Heart Disease Prediction

## Objective
Predict the presence of heart disease using patient medical data.

## Dataset
Heart Failure Prediction Dataset

- Records: 918
- Features: 11
- Target: HeartDisease

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- XGBoost

## Models Used
1. Logistic Regression
2. Random Forest
3. Support Vector Machine (SVM)
4. XGBoost

## Results

| Model | Accuracy |
|---------|---------|
| Logistic Regression | 85.33% |
| Random Forest | 87.50% |
| SVM | 87.50% |
| XGBoost | 86.96% |

## Best Model
Random Forest

## Conclusion
The Random Forest model achieved the highest performance and successfully predicted heart disease with 87.5% accuracy.
